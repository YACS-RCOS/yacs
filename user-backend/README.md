# YACS User Backend
## Description
The backend API for the user management system of YACS Project.

## Getting Started

### Step 0 : Make sure you have docker installed

### Step 1 : clone the repo
```bash
git clone https://github.com/YACS-RCOS/yacs.git
git checkout user_backend
cd ./yacs/user-backend
```

### Step 2 : Run the starting scripts
```bash
./scripts/init.sh
```

### Step 3 : Try it
```bash
curl localhost:5000 
```
